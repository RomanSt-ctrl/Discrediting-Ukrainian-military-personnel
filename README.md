Цей проєкт є модифікованою версією відкритого рішення https://github.com/oasis-open/cti-stix-visualization , адаптованою для візуалізації графу, присвяченого темі «Дискредитація військовослужбовців України».

Інструмент використовується виключно в наукових та дослідницьких цілях: для аналізу інформаційних операцій, взаємозв’язків між об’єктами та структурування даних у форматі STIX 2.x.

Висловлюю щиру подяку розробникам оригінального програмного забезпечення STIX Visualizer від OASIS CTI за їхню роботу, відкритість коду та можливість повторного використання.
